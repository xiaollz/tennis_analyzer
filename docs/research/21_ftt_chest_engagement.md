## 核心概念：胸肌参与 ≠ 肘部往前推

### 关键区分

| | 错误理解 | 正确理解 |
|---|---------|---------|
| **意象** | 肘部往前推 | 胸部往前 press |
| **发力来源** | 肱三头肌/肩部小肌群 | 胸大肌（随躯干转动参与） |
| **感觉** | 手臂在"捅" | 身体在"推"，球是被身体送过去的 |
| **结果** | 手臂发力、动作僵硬 | 大肌群连接、拍面稳定 |

**FTT 教练并不反对"肘部领先"的视觉效果，但反对"用手臂力量推肘部"。** 肘部之所以看起来领先，是因为胸肌通过转体将整个手臂框架带到了前方。

## 三个维度：时机、控制、力量

### 维度一：时机（Timing）
- 胸肌参与**减少自由度**——手臂固定在躯干上，只需控制躯干转动
- 手臂松散时击球窗口只有几毫秒
- 胸肌连接后击球窗口大幅变宽（轨迹更可预测）

### 维度二：控制（Control）
- 手臂是"焊接在躯干上的钢梁"，不是"挂在上面的绳子"
- 拍面像"固定在转盘上的挡板"——转盘转准了，挡板位置就对
- 即使跑动中或站位不佳，胸肌连接在就能控制拍面

### 维度三：力量悖论（Paradox of Power）
- "看起来更僵硬"的动作反而打出更重的球
- 原因：**大脑在感受到控制时才敢释放力量**
- 控制带来自由——胸肌连接给了潜意识安全感，才允许100%速度转体

## 关键认知修正

**"肘部往前推"是危险的意象：**
- 容易被理解为手臂发力 → 增加手臂代偿
- 正确意象：**感受胸部往前 press** → 胸肌连着手臂一起走
- 肘部确实在往前，但动力来源是胸肌+躯干旋转，不是手臂肌肉

**胸肌参与不是一种"发力"，是一种"连接"：**
- 等长收缩或向心收缩，保持手臂-躯干夹角恒定
- 手臂不会滞后于身体（胸肌不滞后）
- 手腕可以自然滞后（手腕lag是好的）
- 胸肌滞后 = 坏的（手臂被甩在身后）
