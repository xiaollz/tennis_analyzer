## 教学理念

Feel Tennis 的核心教学理念：
- **感觉优先**：通过身体感觉学习，而非机械模仿动作
- **生物力学基础**：理解动作"为什么"这样做
- **循序渐进**：从基础到高级，从慢到快

### 最佳学习路径
```
理解原理 → 影子挥拍 → 自抛/喂球 → 合作对拉 → 半场对抗 → 比赛应用
```

### 2.7 正手大师课核心算法 (Tennisnerd x ATP Coach Adri)

> **视频来源**：【Master Your Forehand with ATP Coach Adri - Tips and Common Mistakes】(Tennisnerd 频道，44分钟深度教学)
> **核心针对**：引拍停顿、手臂拉拍过大、击球点不稳定、发力脱节等问题。

**1. 解决“手臂过度引拍”的终极方案**
- **左手主导（Left Arm Lead）**：Adri 极其强调，引拍时右手完全不发力，是靠左手把球拍向后带。
- **视野守则（Field of Vision）**：这是纠正 Bug 的关键——**球拍绝不能离开你的视野**。如果引拍太深（超过肩膀线），大脑就会失去对拍头的控制感，从而导致卡顿或击球点偏后。
- **保持紧凑**：只要球拍还在视野内，就能保证每次挥拍的轨迹是可重复的、简单的，而简单的动作才最稳定。

**2. 修复“别扭感”与增加爆发力：左手拉动（The Pull）**
- **加速秘诀**：当你准备好击球时，不要只想着右手往前推。Adri 提供了一个能让你球速增加 20km/h 的技巧：用左手猛地向后“拉（Pull）”。
- **物理效应**：左手向后拉产生的反作用力会瞬间引爆右手的加速度。这解决了你感觉“脱节”的问题，让身体像一个整体在旋转。

**3. 关于“拍面朝地”与“上旋”的深度校准**
- **Patting the Dog（拍狗狗手势）**：Adri 支持引拍时拍面朝下，因为这能产生更多的 Lag（滞后/鞭打效应），从而带来更强的加速度和上旋。
- **击球瞬间垂直（Flat）**：他特别指出，尽管引拍是闭合的，但**触球那一刻拍面必须是垂直的**。上旋来自于拍头“由下向上”的刷球动作，而不是“盖”着球打。

## 学习建议

### 训练模板（45-60 分钟单主题）

1. **选 1 个指标**（只看一个）：例如"击球点是否在身体前方 20-40cm"
2. **影子挥拍 3-5 分钟**：慢 → 中速；每 5 次停一下检查击球区
3. **自抛/喂球 10-15 分钟**：给自己时间与空间，把动作做对
4. **合作对拉 10-15 分钟**：加入外部约束（只打斜线；必须过网 1m；必须落在后 1/3）
5. **半场对抗 10 分钟**：带规则的小比赛（例如只能打对手反手）
6. **录像 30 秒 + 复盘 2 分钟**：只看你选的那个指标，写一句结论

### 每周训练安排

| 阶段 | 重点 | 时长 |
|------|------|------|
| 热身 | 动态拉伸、步法热身 | 10-15分钟 |
| 迷你网球 | 球感、控制 | 10分钟 |
| 底线练习 | 正手/单反 | 30-40分钟 |
| 发球练习 | 各类发球 | 15-20分钟 |
| 网前练习 | 截击、高压 | 10-15分钟 |
| 对打/比赛 | 综合应用 | 剩余时间 |

### 学习顺序建议

```
基础概念 → 正手 → 单反 → 发球 → 截击 → 步法 → 战术 → 综合
```
