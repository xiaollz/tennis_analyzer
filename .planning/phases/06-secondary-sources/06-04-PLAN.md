    1. Load all concept JSONs from tomallsopp_dir/*.json and feeltennis_dir/*.json
    2. For each concept in all secondary extraction JSONs:
       a. Call classify_concept() against the existing FTT registry
       b. If "agreement": call boost_confidence() on the matched FTT concept ID
       c. If "complement": add to registry with confidence=0.6, source tag = channel name
       d. If "conflict": log to conflict_log with {concept_name, source, matched_ftt_id, reason}

       boost to 0.7 (two-secondary agreement)
    4. Return ReconciliationResult

      "incorrect") AND the matched FTT concept description does NOT contain those words → classify as conflict
    - Otherwise, if fuzzy match >= 70 → agreement
    - If no match → complement
    - Default ambiguous cases to "complement" (safer, per RESEARCH.md recommendation)
