    1. **HypothesisStatus** enum: Literal["active", "confirmed", "eliminated"] -- use a str Enum like the existing pattern.

    2. **ObservationJudgment** enum: Literal["yes", "no", "unclear"] -- str Enum.

    3. **Observation(BaseModel):**
       - id: str (e.g. "obs_r1_01")
       - round_number: int (Field(ge=0))
       - frame: str (e.g. "图3" -- which frame in the keyframe grid)
       - description: str (what was seen)
       - judgment: ObservationJudgment
       - confidence: float (Field(ge=0.0, le=1.0))
       - directive_source: str (which check_step or directive generated this)

    4. **HypothesisUpdate(BaseModel):**
       - hypothesis_id: str
       - action: HypothesisStatus (confirm/eliminate/adjust -- but adjust keeps "active")
       - reason: str

    Wait -- re-read research. The VLM returns hypothesis_updates with action "confirm|eliminate|adjust". Adjust is NOT a status, it means "stay active but change confidence". So HypothesisUpdate.action should be Literal["confirm", "eliminate", "adjust"].

    5. **Hypothesis(BaseModel):**
       - id: str (e.g. "hyp_scooping_active_lag")
       - chain_id: str (DiagnosticChain ID, dc_ prefix)
       - root_cause_concept_id: str (from chain.root_causes)
       - name: str
       - name_zh: str
       - status: HypothesisStatus (default "active")
       - confidence: float (Field(default=0.5, ge=0.0, le=1.0))
       - supporting_observations: list[str] = [] (observation IDs)
       - contradicting_observations: list[str] = []
       - check_steps_completed: list[int] = [] (indices into chain.check_sequence)
       - round_introduced: int (Field(ge=0))
       - round_resolved: int | None = None

    6. **RoundResult(BaseModel):**
       - round_number: int (Field(ge=0))
       - prompt_sent: str (the full prompt text sent to VLM)
       - raw_response: str (raw VLM output)
       - observations: list[Observation] = []
       - hypothesis_updates: list[HypothesisUpdate] = []
       - timestamp: str | None = None (ISO format, optional)

    7. **DiagnosticSession(BaseModel):**
       - session_id: str (e.g. "sess_20260403_143000")
       - video_path: str | None = None
       - image_b64_hash: str | None = None (for dedup/cache, NOT the full b64)
       - hypotheses: list[Hypothesis] = []
       - observations: list[Observation] = []
       - rounds: list[RoundResult] = []
       - active_chain_ids: list[str] = [] (chain IDs under investigation)
       - checked_steps: dict[str, list[int]] = {} (chain_id -> step indices evaluated)
       - convergence_score: float = Field(default=0.0, ge=0.0, le=1.0)
       - max_rounds: int = Field(default=4, ge=1, le=10)
       - final_result: dict | None = None (the root_cause_tree JSON, set at end)
       - created_at: str | None = None (ISO timestamp)
       - completed_at: str | None = None
