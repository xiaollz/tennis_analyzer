    `generate_diagnostic_chains(kg: KnowledgeGraph) -> list[DiagnosticChain]`:

    1. Find all symptom nodes: nodes where category == "symptom" in graph attributes
    2. For each symptom, call kg.get_causal_chain(symptom_id, cause_type="causes")
    3. If causal paths exist:
       a. Collect all root causes (terminal nodes in paths)
       b. Find drills: for each root cause, look for outgoing "drills_for" edges
       c. Build check_sequence: each intermediate node in causal path becomes a DiagnosticStep
          - check: node description (English)
          - check_zh: node name_zh
          - if_true: next node in path (the cause)
          - if_false: None (linear chain) or next check in sequence
       d. Create DiagnosticChain with dc_ prefix ID, priority based on in-degree of symptom node
