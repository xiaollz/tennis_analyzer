       d. Build a lookup dictionary keyed by muscle name (English lowercase)

       Save to _muscle_profiles.json as list of dicts: [{name, name_zh, body_segment, function, training_methods: [], common_failures: []}]

       Save to _concept_muscle_map.json as {concept_id: [{muscle, role, action}]}
